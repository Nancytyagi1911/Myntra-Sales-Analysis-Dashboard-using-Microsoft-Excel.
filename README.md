🛍️ Myntra Sales Analysis Dashboard (Excel Project)

This project focuses on analyzing Myntra product sales data using Microsoft Excel. The goal is to transform raw e-commerce data into meaningful insights using data cleaning, pivot tables, and dashboard visualization.

The dashboard helps understand product performance, brand popularity, revenue trends, and discount impact.

📊 Project Objectives

Clean and prepare raw Myntra product data

Perform data transformation and analysis

Identify top performing brands and products

Analyze revenue and discount trends

Build an interactive Excel dashboard for business insights

🛠️ Tools & Skills Used

Microsoft Excel

Data Cleaning

Pivot Tables

Data Processing

Excel Formulas

Data Visualization

Dashboard Design

📂 Dataset Information

The dataset includes Myntra product information such as:
